# 🩺 Landing Page - Fonoaudiologia Premium

Uma landing page elegante e performática desenvolvida para profissionais de Fonoaudiologia, focada em conversão, acessibilidade e design premium.

---

## ✨ Destaques do Projeto

- 🚀 **Performance Extrema:** Desenvolvido em HTML5 e CSS3 puros para carregamento instantâneo.
- 📱 **Totalmente Responsivo:** Experiência fluida em desktops, tablets e smartphones.
- 🎨 **Design Moderno:** Estética limpa (Glassmorphism), tipografia elegante e micro-animações.
- 🔒 **Segurança:** Configurações de DevOps aplicadas para versionamento seguro.
- 📩 **Formulário Inteligente:** Interface intuitiva com máscaras de campo e validação.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais estável e rápido na web moderna:

- **HTML5:** Estrutura semântica e SEO-friendly.
- **CSS3:** Layouts modernos com Flexbox/Grid e variáveis nativas.
- **JavaScript:** Lógica de interface e animações via `IntersectionObserver`.
- **FontAwesome:** Ícones vetoriais de alta qualidade.
- **Google Fonts:** Tipografia premium (Inter/Outfit).
